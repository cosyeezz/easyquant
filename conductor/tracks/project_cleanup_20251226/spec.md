# Spec: 项目清理与维护

## 目标
通过删除无效文件和优化配置，保持代码库的整洁和轻量化，提高搜索和加载效率。

## 核心需求
1. **缓存清理:** 删除所有的 `__pycache__`, `.pytest_cache`, `.next` (如不需要) 等。
2. **废弃代码清理:** 识别并删除不再使用的 `debug_*.py` 文件、旧版编辑器备份或其他废弃的临时脚本。
3. **日志与临时文件:** 清理 `logs/` 目录下的陈旧日志，清理根目录下的临时文件 (如 `dev_server.log`)。
4. **Git 忽略规则优化:** 完善 `.gitignore` 和 `.geminiignore`，确保上述文件不再被追踪。
