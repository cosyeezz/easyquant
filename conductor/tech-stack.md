# Tech Stack

## Backend
- **Language:** Python (3.10+)
- **Framework:** FastAPI (高性能异步 Web 框架)
- **Database:** PostgreSQL
- **ORM & Migrations:** SQLAlchemy (异步模式) & Alembic
- **Data Analysis:** Pandas
- **Real-time Communication:** WebSocket (用于推送实时日志和事件流)

## Frontend (Easy-Canvas)
- **Framework:** Next.js (TypeScript)
- **Node Editor:** React Flow
- **Styling:** Tailwind CSS (遵循 Linear/Stripe 风格)
- **State Management:** React Context / Hooks
- **Testing:** Vitest (Unit/Integration), React Testing Library

## Infrastructure & Tools
- **Process Management:** 自定义一键启动脚本 (start.sh/stop.sh)
- **Environment Management:** python-dotenv
- **Dependency Management:** pip (requirements.txt) & pnpm/npm
